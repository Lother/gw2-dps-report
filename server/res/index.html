How to Use

  • Get latest token from us.

  • Enable "Save EVTC logs after encounters" option in arcdps.
    Use Alt + Shift + H to show arcdps.
    Use Alt + Shift + T to shot option window.

  • Enable "Compress logs with PowerShell (Win10)" option in arcdps if possible.
    This requires Windows© 10.

  • Upload your .evtc, .evtc.zip, or .zevtc file.

  • That's all. Generated URL would be sent to Discord.

This is a private service to upload arcdps logs, generate reports by Elite Insights, and send to our Discord.

The source code can be found on GitHub. PRs are welcome.

  • A simple HTTP server and written by Rust.
  • A parser script which invokes Elite Insights and send reports to webhooks by cURL.
  • A simple HTML pages which use Bulma and DropzoneJS.
  • A Docker image that runs above programs and serve HTTP endpoints.
  • All static files are served by cdnjs and Cloudflare.
